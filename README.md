# cpp-shopping-cart-system
A console-based Shopping Cart System built using C++ that allows users to add, remove, view items, and generate a final bill using object-oriented programming concepts.

A simple console-based Shopping Cart System developed in C++.
This project simulates an online shopping cart where users can add products, remove products, view cart items, and generate a final bill.

---

## 🚀 Features

- 🛍️ Add items to cart
- ❌ Remove items from cart
- 📋 View cart items
- 💰 Calculate total bill
- 🧾 Generate final receipt
- 🖥️ Menu-driven interface

---

## 🛠️ Technologies Used

- C++
- Object-Oriented Programming (OOP)
- STL Vector
- Functions
- Loops & Switch Case
